## 📱 МИКРОИНТЕРРАКЦИИ

### 1. Анимация добавления в корзину:
```
Этап 1: [  В КОРЗИНУ  ]
        ↓ (нажатие)
Этап 2: [  ⏳ Добавляем...  ] (fade + scale)
        ↓ (1 секунда)
Этап 3: [  ✅ В КОРЗИНЕ  ] (bounce + зеленый цвет)
        ↓ (тост сообщение)
Этап 4: "Товар добавлен в корзину" (появляется снизу)
```

### 2. Переключение изображений:
```
Свайп влево:  [🖼️1] → [🖼️2] (slide transition)
Свайп вправо: [🖼️2] → [🖼️1] (slide transition)
Тап на миниатюру: [🖼️mini] → [🖼️big] (fade transition)
```

### 3. Увеличение изображения:
```
Обычное: [   ИЗОБРАЖЕНИЕ   ]
         ↓ (двойной тап)
Zoom:    [     ИЗОБРАЖЕНИЕ     ]
         (pinch to zoom, pan to move)
```
