## 🔧 Управление оптимизациями

Все оптимизации включаются автоматически при установке зависимостей.
Для отключения измените флаги в `app/products.tsx`:

```typescript
const ENABLE_OPTIMIZATIONS = {
    useFlashList: false,      // Отключить FlashList
    usePagerView: false,      // Отключить PagerView
    useWeakMapCache: true,    // Оставить включенным
    useProgressiveImages: true, // Оставить включенным
    enableMetrics: __DEV__,   // Только в dev
};
```
