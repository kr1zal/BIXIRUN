# Реализация корзины в BIXIRUN

В этом документе описана реализация функциональности корзины для приложения BIXIRUN.

## Архитектура

Реализация корзины включает следующие компоненты:

1. **Redux-слайс для управления состоянием корзины:**
   - `cartSlice.ts` - Redux-слайс с actions и reducers
   - Функции для добавления/удаления товаров и изменения количества
   - Селекторы для расчета общей стоимости и количества товаров

2. **UI-компоненты корзины:**
   - `CartItem.tsx` - компонент элемента корзины
   - `CartSummary.tsx` - компонент с итоговой информацией и оформлением заказа
   - `EmptyCart.tsx` - компонент для отображения пустой корзины

3. **Интеграция с продуктами:**
   - Обновленная страница продукта для добавления товаров в корзину
   - Индикатор количества товаров в корзине в нижнем меню

## Функциональность

### Redux Store

Redux-слайс для корзины (`cartSlice.ts`) предоставляет следующие функции:

- `addToCart` - добавление товара в корзину
- `removeFromCart` - удаление товара из корзины
- `updateQuantity` - изменение количества товара
- `clearCart` - очистка корзины

И селекторы:

- `selectCartTotal` - для получения общей стоимости товаров в корзине
- `selectCartItemsCount` - для получения общего количества товаров

### Страница корзины

Страница корзины (`cart.tsx`) имеет следующие возможности:

- Отображение списка товаров в корзине
- Возможность изменения количества каждого товара
- Удаление товаров из корзины
- Расчет общей стоимости, включая налог и доставку
- Кнопка оформления заказа (пока в режиме заглушки)
- Отображение специального UI при пустой корзине

### Обновленное нижнее меню

Нижнее меню приложения (`FooterNavigation.tsx`) теперь включает:

- Иконку корзины
- Счетчик количества товаров в корзине

## Пользовательский опыт

1. **Добавление товара в корзину:**
   - Пользователь выбирает количество с помощью `QuantitySelector`
   - Нажимает кнопку "Добавить в корзину"
   - Показывается уведомление об успешном добавлении с возможностью перейти в корзину

2. **Управление корзиной:**
   - Пользователь может изменять количество товаров
   - Удалять товары по одному
   - Видеть актуальную информацию о стоимости заказа

3. **Оформление заказа:**
   - Показывается информация о стоимости товаров, доставке и налогах
   - При нажатии на кнопку "Оформить заказ" показывается заглушка
