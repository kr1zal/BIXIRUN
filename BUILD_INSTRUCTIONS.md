## 🚀 ПОШАГОВЫЕ ДЕЙСТВИЯ:

### 1. Скачайте проект на ваш Mac
```bash
# Если проект еще не на вашем компьютере
git clone [URL_ВАШЕГО_РЕПОЗИТОРИЯ]
cd bixirun
```

### 2. Установите зависимости
```bash
npm install
```

### 3. Установите CocoaPods зависимости
```bash
cd ios
pod install
cd ..
```

### 4. Откройте проект в Xcode
```bash
open ios/BIXIRUN.xcworkspace
```
⚠️ **ВАЖНО:** Открывайте `.xcworkspace`, НЕ `.xcodeproj`!

### 5. Настройте в Xcode:

#### A. Выберите ваш iPhone:
- В верхней панели Xcode выберите ваш подключенный iPhone

#### B. Настройте подписи:
1. Выберите проект **BIXIRUN** слева
2. Выберите таргет **BIXIRUN**
3. Вкладка **Signing & Capabilities**:
   ```
   ✅ Automatically manage signing
   Team: [Ваш Apple ID]
   Bundle Identifier: com.yourname.bixirun (измените!)
   ```

### 6. Билд и установка:
```
Нажмите Cmd + R или кнопку ▶️
```

## 📱 ПОСЛЕ УСТАНОВКИ НА ТЕЛЕФОН:

### 1. Разрешите доверие разработчику:
- Настройки → Основные → VPN и управление устройством
- Найдите ваш Apple ID → Доверять

### 2. Разрешите доступ к камере/микрофону:
- При первом запуске нажмите "Разрешить"

### 3. Тестирование:
1. Откройте приложение
2. Перейдите на экран "Таймер" (автоматически откроется настройки)
3. Установите: Подготовка 3 сек, Работа 10 сек, Отдых 5 сек
4. Нажмите "Начать тренировку"
5. Проверьте что:
   - ✅ Камера запускается
   - ✅ Таймер отсчитывает (3→2→1, затем 10→9→8...)
   - ✅ Кнопка "СТАРТ" начинает запись видео
   - ✅ Таймер накладывается на видео
   - ✅ Кнопка "СТОП" сохраняет видео в галерею

```bash

```

- Проверьте разрешения в настройках iPhone
- Убедитесь что Info.plist содержит описания разрешений
